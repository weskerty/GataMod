# `📥 INSTALAR MOD`
> [!CAUTION]
> Estas modificaciones pueden impedir que el bot se pueda actualizar automaticamente con el comando `.update` `.actualizar`

## Simplemente debes Descargar y mover los archivos de este Repositorio a la carpeta del bot.
### Desde Mystic 🔮
Envia esto al Bot en WhatsApp:
```
$ git remote add https://github.com/weskerty/Modstic.git
git fetch https://github.com/weskerty/Modstic.git
git merge https://github.com/weskerty/Modstic.git
```
Luego podras usar `.install` como en el video 

### Desde Gata 🐈
> [!IMPORTANT]
> Gata no tiene ejecutor shell. Para seguir estos pasos deberas mover manualmente [**Este Plugin**](ejecutar.js) a la carpeta plugins de gata bot o ejecutarlo directamente desde la terminal.

Envia esto al Bot en WhatsApp:
```
.ej git remote add https://github.com/weskerty/Modstic.git
git fetch https://github.com/weskerty/Modstic.git
git merge https://github.com/weskerty/Modstic.git
```
Luego podras usar `.install` como en el video 

# Para Actualizar el bot despues del Mod:
Envia esto a tu Bot en WhatsApp:

### Desde Mystic Oficial 🔮
```
$ git pull https://github.com/BrunoSobrino/TheMystic-Bot-MD.git 
```

### Desde Gata Oficial 🐈
```
.ej git pull https://github.com/GataNina-Li/GataBot-MD.git 
```
